## Project Overview

**Aaron Speech Reboot** is an Obsidian vault for AI scene-driven oral English content, focused on rebuilding tech workplace listening and speaking skills. The repository contains markdown notes organized as an Obsidian knowledge graph.

## Key Characteristics

- **Project type**: Obsidian vault (markdown notes, not a codebase)
- **Primary content**: AI-generated or curated oral English scenarios for tech workplace contexts
- **Structure**: Markdown files organized by scene/topic within the Obsidian framework

## Development Context

This repository does not have:
- Build systems, linters, or test runners
- Source code in programming languages
- CI/CD configuration

Work here involves creating, organizing, and editing markdown content — not running builds or tests.
